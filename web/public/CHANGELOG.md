# Changelog

## [1.0.0] - 2025-12-28

### Added

- **Core Engine**: Initial release of the storage engine with `mmap` persistence.
- **API**: REST endpoints for Insert, Get, Query, and Metrics.
- **Real-Time**: WebSocket server with `INSERT` event broadcasting.
- **Config**: TOML-based configuration system.
- **CLI**: Standard binary entry point.
- **Documentation**: Comprehensive guides for Installation, Configuration, and API.
